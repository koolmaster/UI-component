### Important Notes

- Only using font icon, Better effect with outline icon.
- Add property type to choose icon font type. Only one of the following can be selected: `grommet`, `github`,`game`,`feather`,`flatColor`,`devi`,`circum`,`fontAwesome`,`bootstrap`,`box`,`antDesign`,`ionic5`,`material`.
- Get icon name from react-icon, see more at [react-icon](https://react-icons.github.io/react-icons).
- If the URL is empty then use the action prop. If the URL has data, the URL is preferred.
